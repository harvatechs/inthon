# INTHON Language Support

This VS Code extension provides syntax highlighting and file icon support for the **INTHON** programming language (`.inth`), an agent-level programming language designed for AI-native workflows.

## Features

- **Syntax Highlighting**: Supports control keywords, agentic paradigms (`agent`, `goal`, `policy`, `plan`, `remember`, `recall`, etc.), python bridges, strings, comments, and primtive types.
- **Custom File Icon**: Renders `.inth` files in your VS Code workspace with the official INTHON branding.
- **Bracket Matching & Auto-closing**: Smart editing configurations for functions and statements.

## Installation

### Manual Installation
To install the extension locally:
1. Copy the `ide/vscode` folder into your VS Code extensions directory:
   - **Windows**: `%USERPROFILE%\.vscode\extensions\inthon-support`
   - **macOS / Linux**: `~/.vscode/extensions/inthon-support`
2. Restart VS Code.

## Usage
Simply open any `.inth` file (e.g., `hello.inth`) and enjoy syntax highlighting and file icon support.
